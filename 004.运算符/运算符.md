# 运算符

## 算术运算符

```sql
SELECT
    100, 100+1,100*1,100/1,100-1,100%1,100 mod 1,
    100-3.5,100+3.5,100*3.5,100/3.5,100%3.5, 100 mod 3.5
 FROM DUAL;

 /*
Name       |Value   |
-----------+--------+
100        |100     |
100+1      |101     |
100*1      |100     |
100/1      |100.0000|
100-1      |99      |
100%1      |0       |
100 mod 1  |0       |
100-3.5    |96.5    |
100+3.5    |103.5   |
100*3.5    |350.0   |
100/3.5    |28.5714 |
100%3.5    |2.0     |
100 mod 3.5|2.0     |
 */
```

> 分母为 `0`，结果为 `NULL`

### 隐式转换

```sql
SELECT 100 + '1' FROM DUAL;

/*
+-----------+
| 100 + '1' |
+-----------+
|       101 |
+-----------+
*/

-- 无法将 'a' 转换为数字，将其视为0
SELECT 100 + 'a' FROM DUAL;

/*
+-----------+
| 100 + 'a' |
+-----------+
|       100 |
+-----------+
*/

SELECT CONCAT(100,1) FROM DUAL;
/*
+---------------+
| CONCAT(100,1) |
+---------------+
| 1001          |
+---------------+
*/
```
