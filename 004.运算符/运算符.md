# 运算符

## 算术运算符

```sql
SELECT
    100, 100+1,100*1,100/1,100-1,100%1,100 mod 1,
    100-3.5,100+3.5,100*3.5,100/3.5,100%3.5, 100 mod 3.5
 FROM DUAL;

 /*
Name       |Value   |
-----------+--------+
100        |100     |
100+1      |101     |
100*1      |100     |
100/1      |100.0000|
100-1      |99      |
100%1      |0       |
100 mod 1  |0       |
100-3.5    |96.5    |
100+3.5    |103.5   |
100*3.5    |350.0   |
100/3.5    |28.5714 |
100%3.5    |2.0     |
100 mod 3.5|2.0     |
 */
```

> 分母为 `0`，结果为 `NULL`

### 隐式转换

```sql
SELECT 100 + '1' FROM DUAL;

/*
+-----------+
| 100 + '1' |
+-----------+
|       101 |
+-----------+
*/

-- 无法将 'a' 转换为数字，将其视为0
SELECT 100 + 'a' FROM DUAL;

/*
+-----------+
| 100 + 'a' |
+-----------+
|       100 |
+-----------+
*/

SELECT CONCAT(100,1) FROM DUAL;
/*
+---------------+
| CONCAT(100,1) |
+---------------+
| 1001          |
+---------------+
*/
```

## 比较运算符

返回值：`1`为真；`0`为假；其它情况为 `NULL`。

> 注意隐式转换

```sql
SELECT 'A' = 'A', 'A' = 'B', 'A' = NULL, 1 = '1', 1 = 'a', 0 = 'a', NULL = NULL;

/*
+-----------+-----------+------------+---------+---------+---------+-------------+
| 'A' = 'A' | 'A' = 'B' | 'A' = NULL | 1 = '1' | 1 = 'a' | 0 = 'a' | NULL = NULL |
+-----------+-----------+------------+---------+---------+---------+-------------+
|         1 |         0 |       NULL |       1 |       0 |       1 |        NULL |
+-----------+-----------+------------+---------+---------+---------+-------------+
*/
```

- `<=>`：安全等于运算符，安全地判断两个值、字符串或表达式是否相待。

```sql
SELECT 'A' <=> 'A', 'A' <=> 'B', 'A' <=> NULL, 1 <=> '1', 1 <=> 'a', 0 <=> 'a', NULL <=> NULL;

/*
+-------------+-------------+--------------+-----------+-----------+-----------+---------------+
| 'A' <=> 'A' | 'A' <=> 'B' | 'A' <=> NULL | 1 <=> '1' | 1 <=> 'a' | 0 <=> 'a' | NULL <=> NULL |
+-------------+-------------+--------------+-----------+-----------+-----------+---------------+
|           1 |           0 |            0 |         1 |         0 |         1 |             1 |
+-------------+-------------+--------------+-----------+-----------+-----------+---------------+
*/
```

作业：查询 `commission_pct` 字段为 `NULL` 的数据

错误用例：

```sql
SELECT
    last_name,salary,commission_pct
FROM
    employees
WHERE
    commission_pct = NULL
```

改正：

- `WHERE commission_pct <=> NULL`

- `WHERE commission_pct IS NULL`

- `WHERE ISNULL(commission_pct)`
