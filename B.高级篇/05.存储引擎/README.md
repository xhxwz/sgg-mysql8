## InnoDB和MyISAM对比

| 对比项         | MyISAM                                                 | InnoDB                                                       |
| -------------- | ------------------------------------------------------ | ------------------------------------------------------------ |
| 外键           | 不支持                                                 | 支持                                                         |
| 事务           | 不支持                                                 | 支持                                                         |
| 行表锁         | 表锁，即使操作一条记录也会锁住整个表，不适合高并发操作 | 行锁，操作时只锁某一行，不对其它行有影响，适合高并发的操作   |
| 缓存           | 只缓存索引，不缓存真实数据                             | 不仅缓存索引，还缓存真实数据。对内存要求较高，而且内存大小对性能有决定性影响 |
| 自带系统表使用 | 是                                                     | 否                                                           |
| 关注点         | 性能：节省资源、消耗少、简单业务                       | 事务：并发写、事务、更大资源                                 |
| 默认安装       | 是                                                     | 是                                                           |
| 默认使用       | 否                                                     | 是                                                           |
